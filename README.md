# AI LangChain Lab

Repository for experimenting with AI applications built with LangChain.

## Goal

Build small, practical examples that show how to connect language models with prompts, tools, memory, retrieval, and application workflows.

## Planned Ideas

- Basic chatbot with LangChain
- Prompt templates and chains
- Tool calling examples
- Retrieval-Augmented Generation with documents
- API integration examples
- Evaluation and testing examples

## Tech Stack

- TypeScript or Python
- LangChain
- OpenAI API
- Vector databases
- API services

## Status

Initial repository created for portfolio experiments.
